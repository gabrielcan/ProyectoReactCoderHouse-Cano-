Presento una pagina de compras y ventas de productos varios.


Agregue algunos complementos:
_"sweetalert2" para las alertas
_"react-loader-spinner" para los momentos donde estaba cargando la pagina
_ Trabaje con Boostrap5 de react para los diseños.


Inicias la Pagina mostrando la lista de productos, en la NAV, puedes ingresar al carrito por la opción carrito o por icono, tambien puedes navegar por las categorias para encontrar el producto que buscas, ademas tienes la opcion de checkout.
Una vez que eliges un producto precionando sobre el mísmo puedes ver los detalles del mismo, donde podras elegir la cantidad que quieres comprar y agregarlo al carrito, para luego tener la opcion de continuar comprando o terminar la compra, donde podras ver el formulario para completar la compra, tambien puedes ir directamente al icono de carrito para ver los productos agregados, donde podras borrar alguno de los productos elegidos si no deseas realizar la compra.
Una vez que decides realizar la compra, completas el formulario, donde tendras que completar todos los campos y ademas los campos email deben ser iguales para poder concretar la compra.
